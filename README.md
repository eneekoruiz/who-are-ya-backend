# Who Are Ya? - Backend

Backend for the Who Are Ya? football guessing game.

## What it includes

- Express server and routing
- MongoDB models and controllers
- public player and solution endpoints
- admin authentication and CRUD routes
- seed script for sample data

## Stack

- Node.js
- Express
- MongoDB
- EJS
